## ブランチ

**main**
メインのブランチ。ここがそのまま外部に公開されるので、直接push禁止ルールに設定してある。

**develop**
開発用ブランチ。

**writing**
記事作成用ブランチ。記事書いたらこのブランチでコミットしたあとプッシュ。プルリク出してもらったあと一人にレビュー/承認してもらって記事公開。
